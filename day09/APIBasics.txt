HTTP Status code:
----------------

1XX ->Information
2XX-->indicates the server response
3XX-->indicates Redirected
4XX-->Client side error
5XX-->sever side 

Success
200 ok -> Get ()
201 -->post
204  -->Delete /Patch -->No content


400 -->Bad request
401 -->unauthorized
403 -->forbidden
404-->not found


500-->internal server error
501-->request is not implemented in the server
502-->bad gateway
503-->service unavailable
505-->Gateway timeout
